# 🏛️ BISU-AACCUP Management System
**A centralized platform for Bohol Island State University faculty and staff to track accreditation status and document feedback.**

---

## 🚀 Project Overview
The **BISU-AACCUP System** is designed to modernize the traditional accreditation process. It serves as a central repository for accreditation documents, allowing Deans, Focal Persons, and Faculty to collaborate, provide feedback, and monitor progress in real-time.

### ✨ Key Features
* **Role-Based Dashboards:** Specific interfaces for Deans, Focal Persons, and Staff.
* **Document Management:** Easy tracking of accreditation folders and files.
* **AI-Ready Infrastructure:** Modular design allowing for future AI-assisted feedback.
* **Responsive UI:** A modern, "glass-morphic" design for better user experience.
* **Database Auto-Repair:** Built-in scripts to handle schema synchronization.

### 🛠️ Tech Stack
* **Backend:** PHP (XAMPP / MariaDB)
* **Frontend:** React (for Admin interfaces) and Bootstrap/Tailwind for the landing page.
* **Version Control:** Git & GitHub
