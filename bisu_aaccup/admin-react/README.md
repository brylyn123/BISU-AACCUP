# BISU AACCUP Accreditation Management System

A comprehensive web-based repository and management system designed for **Bohol Island State University - Candijay Campus**. This system streamlines the AACCUP accreditation process by managing documents, reports, and user assignments.

## 🛠️ Project Structure
* **Backend:** PHP with MySQL/MariaDB (XAMPP).
* **Frontend Dashboard:** React + Tailwind CSS (located in `/admin-react`).
* **Icons:** Lucide React.

## 🚀 Getting Started

### 1. Backend & Database Setup (PHP)
1. Ensure **XAMPP** is installed and running (Apache & MySQL).
2. Move the project folder to `C:/xampp/htdocs/bisu_aaccup/`.
3. Create a database named `bisu_aaccup` in PHPMyAdmin.
4. Import the provided `.sql` file from the `/database` folder.

### 2. Admin Dashboard Setup (React)
