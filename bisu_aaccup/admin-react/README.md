# BISU Admin Dashboard (React + Tailwind)

Quick scaffold for a modular React admin dashboard using Tailwind CSS and Lucide icons.

Getting started:

1. cd into `admin-react`
2. Install deps:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

Notes:
- Uses Vite + React + Tailwind. Icons via `lucide-react`.
- Integrate session/auth server-side as needed (this is a frontend scaffold).
